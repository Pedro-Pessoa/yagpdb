Stickyroles

This will prevent people from leaving the server and rejoining just to clear their roles.

What stickyroles does is reassign the roles the member had when they left the server to them again when they join back.



{{checkbox "StickyrolesEnabled" "StickyrolesEnabled" `Reassign roles upon rejoining` .Autorole.StickyrolesEnabled}}

If the member has both a required role and a blacklisted role, sticky roles won't work for them.
It's best practice to use either blacklisted roles or required roles, not both at the same time.
Using both at the same time will work, if used properly though.
Join the support server if you need help.

Perform a retro-active full scan

YAGPDB already performs a scan of your server every minute, but this only includes online/cached members, it would take up too many resources to keep absolutely all members in the state.

Therefore you can use this to perform a full scan of your server, this may become an automatic premium option at some point.

You will have to wait a couple minutes in-between each completed scan, to cancel this you can set the role to "None" and it should stop within a minute.
